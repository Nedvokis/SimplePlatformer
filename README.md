# SimplePlatformer

2D-платформер на Rust. Игрок управляет персонажем, который прыгает по платформам и должен добраться до выхода с уровня.

## Об игре

Классический платформер с видом сбоку. Цель — пройти уровень, перепрыгивая по платформам и избегая препятствий, чтобы добраться до выхода.

### Главное меню

При запуске игры открывается стартовое меню:

| Пункт | Описание |
|-------|----------|
| **Начать игру** | Запуск первого уровня |
| **Настройки** | Громкость, разрешение экрана, управление |
| **Выйти** | Закрыть игру |

Навигация по меню — `↑` / `↓`, выбор — `Enter`.

### Управление

| Клавиша | Действие |
|---------|----------|
| `A` / `←` | Движение влево |
| `D` / `→` | Движение вправо |
| `Пробел` | Прыжок |
| `Esc` | Меню |
