## Порядок реализации

1. Зависимости (avian2d, serde, ron)
2. Состояния игры (GameState enum)
3. Главное меню (Начать / Настройки / Выйти)
4. Экран настроек (громкость, разрешение, полный экран)
5. Физика (avian2d, гравитация, пауза)
6. Игрок (движение, прыжок, детекция земли, смерть)
7. Камера (следование за игроком)
8. Загрузка уровней (RON-файлы, тайлы, выход)
9. Экран паузы (оверлей, Продолжить / Настройки / В меню)
10. Фикс камеры (рендер для UI)
11. Полировка (clippy, ручной тест)
